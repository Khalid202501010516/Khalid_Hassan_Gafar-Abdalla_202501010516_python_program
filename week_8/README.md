# Week 8 Tutorial – Food Delivery System

This Python project generates a receipt for a simple food delivery order.

## Files

- `main.py` – Runs the main program.
- `customer.py` – Collects customer and order information.
- `receipt.py` – Calculates the total and prints the receipt.

## Charges

- Service charge: 5% of subtotal
- Delivery charge: RM5.00 when delivery is selected
- No delivery charge when delivery is not selected

## Running the Program

Run the following command:

```bash
python main.py
